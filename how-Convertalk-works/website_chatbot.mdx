1. To create your Website chatbot , you first need to visit the  Convertalk Dashboard

![5 Website Chatbot Convertalk Resource Centre 1 Pn](/images/how-botpenguin-works/create-newchatbot/5.website_chatbot__Convertalk_resource_centre-1.png)

2. To start creating your new chatbot, click **"Create New Bot." **Select the primary purpose for creating your bot and proceed further.

![5 Website Chatbot Convertalk Resource Centre 2 Pn](/images/how-botpenguin-works/create-newchatbot/5.website_chatbot__Convertalk_resource_centre-2.png)

3. Select the platform: “**Website Chatbot**”

![5 Website Chatbot Convertalk Resource Centre 3 Pn](/images/how-botpenguin-works/create-newchatbot/5.website_chatbot__Convertalk_resource_centre-3.png)

4. You will see a notification: "**Bot has been successfully created.**” Your bot has been successfully created. Then you will land up on this page, now you can configure the below mentioned items using the below links:
